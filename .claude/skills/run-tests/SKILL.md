테스트를 실행하고 결과를 요약하세요.

인수: $ARGUMENTS

## 절차

1. 인수가 있으면 해당 파일만 실행하세요:
   ```bash
   npx vitest run $ARGUMENTS
   ```
   인수가 없으면 전체 테스트를 실행하세요:
   ```bash
   npm test
   ```

2. 결과를 아래 형식으로 요약하세요:
   - 총 테스트 수 / 통과 / 실패
   - 실패한 테스트가 있으면: 파일명, 테스트명, 에러 메시지
   - 모두 통과하면: "모든 테스트 통과" 한 줄로 끝내세요.
